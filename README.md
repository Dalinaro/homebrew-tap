# Homebrew tap for TT Vpn

A simple menu-bar VPN client for macOS — OpenConnect, OpenVPN & IPsec.
Main project: https://github.com/Dalinaro/ttvpn

## Install
```sh
brew install --cask Dalinaro/tap/ttvpn
```
On first launch the app is unsigned, so right-click it in /Applications and choose **Open**.
Then in the app: menu bar → **Open settings… → Install engines** (one admin password).

## Updating
The app updates itself (menu → “update automatically”).
